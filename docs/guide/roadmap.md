# Roadmap

This roadmap helps adopters evaluate direction and release intent.

## Current focus (Q2 2026)

- Stabilize onboarding and npm metadata for public adoption.
- Keep Vue 3.5+ and Nuxt 4 integration examples up to date.
- Expand component docs with copy-paste recipes for common admin workflows.

## Near-term milestones

### M1 - Adoption baseline

- Public npm-first install documentation complete.
- License and metadata consistency complete.
- Latest release published and reflected on npm `latest`.

### M2 - Conversion improvements

- Comparison page and migration notes finalized.
- Compatibility matrix maintained in docs and README.
- One production-like demo URL linked from docs and README.

### M3 - Discovery and community

- awesome-vue / awesome-nuxt listing submissions ready.
- Reusable Reddit/Discord educational post templates prepared.
- Weekly KPI tracking loop active.

## Looking ahead

- Better Nuxt ergonomics (module/global registration options).
- Higher-level recipe blocks for dashboard layouts.
- Expanded i18n and theme authoring utilities.

## Release cadence

- Small, frequent releases are preferred over large batches.
- Every release should include changelog updates and publish verification.
